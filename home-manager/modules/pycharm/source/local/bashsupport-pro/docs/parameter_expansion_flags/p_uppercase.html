${(P)name} interprets the value of name as further parameter name and then evaluates it.

Links

  • Zsh manual on parameter expansion flags

Examples

Using parameter name indirection
declare -A data=([key]=value [key2]=value2)
name=data
# Prints array 'data'
echo ${(P)name}
# Prints value of `key2` or array `data`
echo ${${(P)name}[key2]}
Output: 
value2 value
value2
