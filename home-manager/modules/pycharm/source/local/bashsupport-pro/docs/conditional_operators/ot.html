FILE -ot FILE (conditional operator)

true if file1 is older than file2, based on the modification date.

-ot is not included in POSIX. Avoid it if you want to stay compatible with POSIX shells.

Links

  • Bash manual
  • POSIX specification of test

Examples

Different ways to test if a file is older than another file:
test ~/file1.txt -ot ~/file2.txt && echo "file1 is older"
[ ~/file1.txt -ot ~/file2.txt ] && echo "file1 is older"
# Bash
[[ ~/file1.txt -ot ~/file2.txt ]] && echo "file1 is older"
