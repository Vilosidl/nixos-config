arg1 -ge arg2 (conditional operator)

Arithmetic comparison: true if integer arg1 is greater than or equal to integer arg2.

Links

  • Bash manual
  • POSIX specification of test

Examples

Different ways to compare two integers:
test "$n" -ge 42 && echo "$n >= 42"
[ "$n" -ge 42 ] && echo "$n >= 42"
# Bash
[[ "$n" -ge 42 ]] && echo "$n >= 42"
