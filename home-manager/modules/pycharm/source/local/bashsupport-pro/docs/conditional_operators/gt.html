arg1 -gt arg2 (conditional operator)

true if integer arg1 is greater than integer arg2.

Links

  • Bash manual
  • POSIX specification of test

Examples

Different ways to compare two integers:
test "$n" -gt 42 && echo "$n > 42"
[ "$n" -gt 42 ] && echo "$n > 42"
# Bash
[[ "$n" -gt 42 ]] && echo "$n > 42"
